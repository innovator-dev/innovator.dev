Open Data Portal

Business requirements

Today's society is highly connected, constantly interacting with their smart devices and sensors. In this context, making data publicly available contributes to the growth of a community, organization, municipality and creates seamless data experience where users or citizens can successfully access data, interpret these findings, and utilize key information and insights to make informed operational decisions.

Through Open Data Portal, organizations and institutions can easily manage and publish data, leveraging the power of Cloud technologies and APIs.

Feature highlights

  • supports all types of file formats
  • documents securely stored in a Cloud environment
  • dynamically enables visualization/preview for PDFs and Excel documents, rendering geolocation-based elements on a map
  • API driven platform, enabling interoperability
  • permission-based access control
  • optimized for mobile devices
